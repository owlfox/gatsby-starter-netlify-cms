# ADT, Abstract Data Type
用來寫物件導向程式的文件手法之一，去除語言實作細節。
描述一個 程式/物件 的輸入/輸出/狀態 等等等...
```
ADT List
Data
  insert, pop O(1), some other descriptions... 
Operation
  New
    init condition
    how it works
  Delete
    ...
endADT
```
